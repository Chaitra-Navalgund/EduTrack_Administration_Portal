EduTrack Administration Portal

This repository contains a Django-based student enrollment and administration portal.

GitHub Pages can only host static files. A Django project needs a Python server, installed dependencies, database access, and backend execution, so the full application cannot run directly on GitHub Pages.

The 404 happened because GitHub Pages looked for a file named index.html at the site root and your repository did not have one.

If you want the actual Django app online, deploy it to a platform that supports Python, such as Render, PythonAnywhere, Railway, or a VPS.
